# Quick Deploy Commands

## Deploy Preview (Fast Testing)

```bash
netlify deploy
```

This creates a preview URL you can share and test on mobile devices.

## Deploy to Production

```bash
netlify deploy --prod
```

## Tips

- Preview deploys are fast and perfect for testing
- The preview URL stays active until you deploy again
- You can test on any device with the preview URL
- All Netlify Functions work automatically in previews
